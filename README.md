# Python Basics and Exercises

This repository is dedicated to my learning journey with Python. It contains basic exercises, fundamental programming concepts, and simple console games that I built while learning the language.

## 🛠️ Concepts & Topics Covered
* **Basic Syntax:** Variables, loops, and conditional statements.
* **Data Structures:** Working with dictionaries (`dict.py`), lists, and file handling.
* **Object-Oriented Programming (OOP):** Creating and managing classes (`class.py`).
* **Game Development:** Simple projects like Snake (`snake.py`) and other interactive games.

## 🚀 Technologies Used
* **Python 3**
* **Git & GitHub**
